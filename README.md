# Go Vercel

A simple Go server running on Vercel infra!

## Set it up locally

- Clone the repo

```bash
mkdir ~/Dev/go-vercel -p
cd ~/Dev/go-vercel
git clone https://github.com/arvind-4/go-vercel .
```

- Install dependencies

```bash
cd ~/Dev/go-vercel
go mod tidy
```

- Run the local dev server

```bash
go run main.go
```
